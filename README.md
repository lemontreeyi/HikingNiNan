# element-plus-vite-starter

项目前置：安装nodejs>=16

## Project setup

```bash
npm install
```

### Compiles and hot-reloads for development

```bash
npm run dev
```

### Compiles and minifies for production

```bash
npm run build
```

## Usage

```bash
git clone https://github.com/lemontreeyi/HikingNiNan.git
cd HikingNiNan
npm install
npm run dev
```
